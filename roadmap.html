🏗 Project 1 — Cooperative Multi-Agent Debate System PUBLISHABLE ›
What you build: 3 LLM agents debate a topic. A judge agent scores argument quality. PPO trains agents to improve debate strategy over time. Agents develop emergent specialization — attacker, defender, mediator.
Why This Is Powerful
  • Combines your existing LLM skills + new RL knowledge — zero wasted work
  • Directly relevant to RLHF (how ChatGPT is trained)
  • Publishable as a NeurIPS/ICLR workshop paper if done rigorously
  • Shows PhD committees you can do independent research
Tech Stack
  • Python + PyTorch — you already know this ✓
  • Gymnasium — custom multi-agent environment
  • Stable Baselines3 PPO — training loop
  • Groq API / OpenAI API — LLM agents
  • Weights & Biases (free) — experiment tracking, essential for research
Timeline
  • April: Build basic environment, 2-agent setup
  • May: Add PPO training, run first experiments
  • June: Scale to 3 agents + judge, measure emergent behaviors
  • July: Write up results, clean code for GitHub

PHASE 03 PhD Program 2027 → 2032 +
📅 Year 1 — 2027–28 SETTLE IN ›
  • Survive coursework — typically 3–4 courses per semester
  • Rotate through 2 labs — pick your advisor very carefully
  • Start TA work — this pays your stipend ($1,800–2,200/month)
  • Begin first research project with advisor
  • Attend lab meetings, read 3 papers/week minimum
  • Submit one paper to a top venue (NeurIPS / ICML / ICLR)
Advisor choice is the most important decision of your PhD. Their research direction = your career direction. Do not pick based on prestige alone — check if their students graduate and get good jobs.
