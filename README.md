## About Me

I'm an **AI/ML Engineer** and final-year CS (AI & ML) student building intelligent systems at the intersection of **Reinforcement Learning**, **Multi-Agent coordination**, and applied AI engineering.

My current research focuses on **cooperative multi-agent debate systems** using PPO, human-in-the-loop orchestration, and scalable MARL architectures. I presented my work on adaptive multi-agent orchestration for automated code generation at **NGAISL2026 (HRIT University)**.

Alongside research, I build production AI systems and automation tools through **TriviLabs** — a web & AI agency serving Delhi NCR businesses.

**Goal:** Direct B.Tech → PhD in Reinforcement Learning & Multi-Agent Systems.
